執行2次，animation-direction:執行的方向不同。


animation-direction: normal; 0 -> 600 *2
animation-direction: reverse; 600 -> 0 *2
animation-direction: alternate; 0 -> 600 -> 0
animation-direction: alternate-reverse; 600 -> 0 -> 600

執行1次，animation-fill-mode:結束後停的位置不同。


animation-fill-mode:none;
animation-fill-mode:forwards;
